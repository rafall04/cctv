# frontend/src/components/landing/.module_map.md

## Core Surfaces

- `LandingNavbar.jsx`: public full-mode top navigation, brand link, layout mode toggle, and theme switch.
- `LandingPageSimple.jsx`: compact public landing shell for simple mode.
- `LandingHero.jsx`: public hero surface with sanitized area coverage HTML and stats bar.
- `LandingPublicTopStack.jsx`: top banner stack for event and announcement surfaces.
- `LandingDiscoveryStrip.jsx`: capped discovery tabs/cards for live/top/area/new public lists.
- `LandingQuickAccessStrip.jsx`: compact favorite/recent access strip for public cameras.
- `LandingSmartFeed.jsx`: compact ranking-based public feed sections.
- `LandingCamerasSection.jsx`: public workspace for grid/map/playback modes and search/filter controls.
- `LandingResultsGrid.jsx`: progressively mounted public camera cards with device-aware windows.
- `LandingMobileDock.jsx`: mobile-only bottom dock for public Home/Map/Grid/Favorit/Playback navigation; callback mode serves landing pages and href mode serves public playback route links; page shells must suppress it while popup or multi-view video surfaces are active.

## Shared Helpers

- `../../utils/publicLandingSections.js`: pure discovery, quick-access, and responsive grid window helpers.
- `../../utils/sanitizePublicHtml.js`: allowlisted sanitizer for public HTML snippets from settings.
- `../../utils/areaPublicRanking.js`: pure public area ranking helpers for live, top, newest, and related lists.
