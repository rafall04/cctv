# frontend/src/components/admin/playback-tokens/.module_map.md

## Components

- `PlaybackTokenForm.jsx`: Create-token form, selected-camera rule controls, and create-flow camera name/id/area filter UI.
- `PlaybackTokenSharePanel.jsx`: Displays generated share text and exposes guarded Copy, native Share, and WhatsApp actions.
- `PlaybackTokenTable.jsx`: Active token table, edit-token policy controls, repeat-share action, session clear, revoke action, and edit-flow camera name/id/area filter UI.

## Data Flow

- Page state comes from `../../../hooks/admin/usePlaybackTokenManagementPage.js`.
- Create form receives `visibleCreateCameras`, selected camera IDs, and create search state from the hook.
- Edit table receives `visibleEditCameras`, selected edit camera IDs, and edit search state from the hook.
- Share panel never builds share text itself; it only renders and dispatches the hook-provided text.
