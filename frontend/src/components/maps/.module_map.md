# frontend/src/components/maps/.module_map.md

## Components

- `MapTopChrome.jsx`: top-left public map area filter, reset button, and decorative zoom hint. Decorative hints must keep `pointer-events-none`; only concrete controls opt into `pointer-events-auto`.

## Utilities

- `../../utils/mapAreaSummary.js`: Pure area aggregate grouping, status counts, and anchor selection used by `MapView.jsx`.
- `../../utils/mapMarkerLayout.js`: Pure stacked marker display coordinate offsets used by `MapView.jsx`.
