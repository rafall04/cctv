# backend/utils/.module_map.md

## Utilities

- `authCookieOptions.js`: request-aware HttpOnly auth cookie option helper for login and refresh responses, including direct-IP and HTTPS proxy behavior.
- `cameraDelivery.js`: camera delivery/source profile normalization helpers.
- `cameraProjection.js`: camera read model/projection helpers.
- `internalIngestPolicy.js`: internal ingest policy helper functions.
- `thumbnailStrategyPolicy.js`: camera thumbnail capture strategy normalizer used by camera writes and thumbnail generation.

## Verification

- Auth cookie helper: `npm test -- authCookieOptions.test.js`.
- Camera utility helpers: run their matching backend test files plus impacted service tests.
